Work · 01 of 05 · 2025

SaaS platform for Shipeezi

Rebuilding a fragmented logistics platform around one token-based, multi-brand design system — 200+ components, light & dark, web & mobile.

Senior Product Designer 10 months Supply chain · SaaS 12 core flows

The challenge

Bootstrap here, Material UI there — and no real structure anywhere.

Designers reused frames instead of components. Developers implemented the same UI element three different ways. Every new feature added more visual debt.

The mandate: replace the legacy foundation with a system serving two brands, twelve core flows and a fast-moving engineering team — without pausing the roadmap.

The work

Token-based foundation

200+ atomic components in Figma, synchronized with Tailwind CSS and ShadCN — light/dark modes and multi-brand theming out of the box.

Smart tables & Kanban

Multi-select filters, search, saved views and visual tagging — plus a drag-and-drop Kanban with customizable statuses for workflow-heavy ops.

The outcomes

0+Components
0%Faster delivery
0Brands, one system
0Flows redesigned
  • Unified visual language across brands and platforms — web and mobile.
  • Legacy Bootstrap/Material clutter eliminated from the codebase.
  • Fully tokenized for dark mode and future expansion.
  • Onboarding completion time improved 25%; satisfaction up 30% in testing.

“It's the first time I haven't had to guess what a component should look like.”

Frontend engineer, Shipeezi
